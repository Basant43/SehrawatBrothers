# 🚀 QUICK START GUIDE - 5 MINUTES TO LIVE PORTFOLIO

## Step 1: Download & Extract (1 minute)
```
1. Download: basant-mehto-portfolio.zip
2. Right-click → Extract All
3. Choose destination folder
```

## Step 2: View Locally (1 minute)
```
1. Go to extracted folder
2. Find "index.html"
3. Double-click to open in browser
4. ✅ Your portfolio is live locally!
```

## Step 3: Create GitHub Account (2 minutes)
```
1. Go to github.com
2. Click "Sign up"
3. Use email and create password
4. Verify email
```

## Step 4: Create Repository (1 minute)
```
1. Click "+" → "New repository"
2. Name: basant-mehto-portfolio
3. Select "Public"
4. Click "Create repository"
```

## Step 5: Upload Files to GitHub
### Method A (Easiest - Web Upload):
```
1. Click "Add file" → "Upload files"
2. Drag all portfolio files
3. Click "Commit changes"
4. Wait 1 minute
5. Go to Settings → Pages
6. Your site is at: 
   https://YOUR_USERNAME.github.io/basant-mehto-portfolio
```

### Method B (Using Git):
```bash
cd extracted-portfolio-folder
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/basant-mehto-portfolio
git branch -M main
git push -u origin main
```

Then enable Pages in Settings.
