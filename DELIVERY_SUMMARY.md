## 🐛 TROUBLESHOOTING

### Videos not playing?
- Check files are in `videos/` folder
- Ensure filename matches (case-sensitive on Linux)
- Try different browser
- Check video format is MP4

### Slow loading?
- Videos are large (normal)
- Use local server for best experience
- Check internet connection

### Mobile layout broken?
- Clear browser cache
- Try different browser
- Check device has minimum screen width

### Navigation not working?
- JavaScript might be disabled
- Check browser console for errors
- Try different browser
